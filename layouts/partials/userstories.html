User Stories

Using the How Might We statements, I created user stories in order to brainstorm the best approach to take for the development of the product. This step was crucial for exploring ways to create seamless experiences for the user.

“Epics,” which were overarching stories based on the problem statements, were explored in detail by finding step-by-step solutions for them. These solutions were then expanded on in order to get a more holistic view of the possible solutions.
