Search

The search page is a main feature for Seed & Soil that will be used frequently, so it was important to nail down a workable design for the aspect of the app. 

  • Similar to the dashboard, the overlays on the images were blurred to make the text more legible and meet the WebAim criteria.
  • I added a horizontal line above the items in the list so that they could fade into it as the user scrolls. 
  • The spacing between the list items was adjusted to equal lengths.
